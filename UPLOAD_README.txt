 GitHub アップロード手順
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【重要】この「deploy」フォルダを「丸ごとアップ」すると
deploy/ というサブフォルダができてしまい正しく動きません。
フォルダの「中身だけ」をリポジトリのルートに置いてください。

━━━ GitHub Web の手順 ━━━

1. GitHubリポジトリを開く
2. 右上「Add file」→「Upload files」をクリック
3. このフォルダを開いて「中のファイルを全選択」
   ※ Macなら Cmd+A、Windowsなら Ctrl+A で全選択
4. 選択したファイルをブラウザにドラッグ＆ドロップ
   ※「deploy」フォルダごとではなく、中のファイルを選んで！
5. 「Commit changes」で保存
6. Actions タブでビルド完了（✓）を確認
7. 下記URLで確認→「v2.11」が表示されれば成功！

━━━ アップロード確認 ━━━

   https://clmiwahotline-rgb.github.io/v.html

「v2.11 ✓ アップロード成功」が出ればOK。
出なければファイルがサブフォルダに入った可能性あり。

━━━ スマホで緑ヘッダーを確認 ━━━

1. まずこのURLをブラウザで開く：
   https://clmiwahotline-rgb.github.io/モバイル.html?new=1
2. 「業務メニュー」の上に
   「緑バー＋AA・人・PCの3つのボタン」が出れば成功！
3. ホーム画面に追加済みの場合：
   一度削除して、ブラウザから再度「ホームに追加」してください
   （PWAキャッシュが残るため）
